Blog RAG API

This is the API for the RAG model. Use the endpoints below to interact with it.

Endpoints

GET /api/health

Health check endpoint to verify the API is working

Example: /api/health

POST /api/chat

Chat with the RAG model using chat format

{
  "messages": [
    {"role": "system", "content": "Your system prompt here"},
    {"role": "user", "content": "User message here"}
  ],
  "model": "llama3.2:3b"
}

Example Usage

fetch('/api/chat', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        messages: [
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": "Tell me about geocaching."}
        ],
        model: "llama3.2:3b"
    }),
})
.then(response => response.json())
.then(data => console.log(data));
